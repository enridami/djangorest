# 🧑‍💼 Sistema de Recursos Humanos (RH)

Aplicación web full stack para la gestión de empleados.

## Stack
- Frontend: React + Vite
- Backend: Django REST Framework
- DB: MySQL
- Axios + Bootstrap

## Funcionalidades
- Listar empleados
- Agregar empleado
- Editar empleado
- Eliminar empleado

## API
http://localhost:8080/api/empleados/

## Endpoints
- GET /api/empleados/
- GET /api/empleados/{id}/
- POST /api/empleados/
- PUT /api/empleados/{id}/
- DELETE /api/empleados/{id}/

## Frontend
npm install
npm run dev

## Backend
python manage.py runserver 8080
